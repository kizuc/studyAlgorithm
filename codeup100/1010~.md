
## 1010. 정수 1개 입력받아 출력

```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		
		System.out.println(n);
		
	}

}
```

## 1011. 문자 1개 입력받아 출력

```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		
		Scanner sc = new Scanner(System.in);

		char x = sc.next().charAt(0);	// Scanner에는 char 받는 게 없다.
										// charAt(int index)
		System.out.println(x);
		
	}

}
```

## 실수 1개 입력받아 그대로 출력하기

```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		
		Scanner sc = new Scanner(System.in);
		
		float x = sc.nextFloat();
		System.out.println(x);
		
	}

}
```
